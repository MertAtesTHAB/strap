### Overview
STRAP is a prescriptive framework and software prototype for dynamic KPI prioritization and automated action planning in Sales, Marketing and Innovationmanagement. Developed as part of a Bachelor Thesis.

### Key Features
* **Dynamic Weighting:** Adjusts KPI importance based on company size and growth strategy.
* **Prescriptive Logic:** Links performance deviations directly to a database of strategic actions.
* **Mathematical Core:** Utilizes a multi-level 100-point distribution algorithm for objective scoring.
* **Visual Output:** Generates Radar Charts for immediate gap analysis.

### Academic Context
This project was developed by **Mert Ates** and **Prof. Dr.-Ing. Alexander Czinki** as part of a Bachelor Thesis. The goal was to invent a new framework that helps prioritizing KPI´s in industrial companies focussing on Sales, Marketing and Innovationmanagement.
